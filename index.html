基于NLP的法语新闻推荐系统

  • 项目简介：这是一个基于自然语言处理（NLP）技术的新闻推荐系统，能够根据用户的阅读历史和偏好，为用户推荐最相关的法语新闻。该系统是为了提高公司的新闻推荐应用的用户使用相关指标而开发的，目标用户是法语地区的新闻爱好者。该系统使用了微软的MIND数据集和NRMS模型论文作为参考，采用了Transformer模型作为核心算法，实现了高效的新闻表示和匹配。
  • 我的角色：在这个项目中，我是主要的研发人员，负责从数据处理到模型部署的整个流程。我与其他团队成员密切合作，共同完成了项目的需求分析，设计，开发，测试和优化。
  • 技术栈：
    • PyTorch | Transformer | Docker | Kubernetes | RESTful API | Git | Linux
  • 主要职责：
    • 负责对接业务数据接口，将原始的新闻数据和用户数据转换为模型可处理的数据格式，进行数据清洗，特征构建和特征筛选等工作。
    • 负责利用现有的NLP模型，如BERT，训练并生成基于法语的词向量文件，用于表示新闻和用户的文本信息。
    • 负责构建基于Transformer的新闻推荐模型，使用小型数据集进行训练和调试，并使用GPU云平台进行完整的训练和评估。
    • 负责将训练好的模型封装为Docker容器，并将其部署到Kubernetes分布式服务架构中，提供RESTful API接口，实现新闻推荐的在线服务。
    • 负责运行并测试模型在实际场景中的表现，收集用户反馈和数据指标，针对不足持续改进模型的性能和效果。
  • 项目成果：通过这个项目，我学习并掌握了NLP和推荐系统的相关知识和技术，如Transformer模型，词向量，新闻表示和匹配等。我也提高了我的软件开发能力，如数据处理，模型构建，容器化部署，分布式服务等。该项目为公司的新闻推荐应用提供了一个高效的法语新闻推荐模块，显著提高了用户的阅读量，点击率，留存率等关键指标，达到了项目的预期目标，获得了公司和用户的好评。

强化学习学生能力测试系统

  • 项目简介：这是一个基于强化学习技术的题目选择算法的研究项目，旨在有效地评估考生的知识状态。该项目是为了解决现有的基于信息量的题目选择算法存在的两个问题而开展的，这两个问题分别是：一是信息量方法无法捕捉题目之间以及知识点之间的隐含的认知信息，二是信息量方法依赖于考生的估计的知识状态，而这些状态本身是不准确的。该项目提出了使用强化学习技术来自动地学习题目选择算法的方法，能够合理地捕捉题目之间的认知关系，避免不必要的题目测试，而且不依赖于考生的估计的知识状态。
  • 我的角色：在这个项目中，我是核心的算法研发人员，负责整个算法部分的程序设计，数据处理，代码编写和调试。我与其他团队成员紧密合作，共同完成了项目的创意，设计，实验和论文撰写。
  • 技术栈：
    • Pytorch | Numpy | Pandas | Scikit-learn | Git | Linux
  • 主要职责：
    • 负责参与项目的创意讨论和创新，提出了使用强化学习技术来改进题目选择过程的想法，并与团队成员一起设计了算法的框架和流程。
    • 负责完成整个算法部分的程序设计，数据处理，代码编写和调试，使用TensorFlow框架实现了基于强化学习的题目选择算法，并与现有的基于信息量的题目选择算法进行了对比和分析。
    • 负责使用模拟的考生数据和真实的考生数据进行了大量的实验，验证了算法的有效性和优越性，并使用Numpy，Pandas，Scikit-learn和Matplotlib等工具进行了数据处理和可视化。
    • 负责与团队成员一起撰写了学术论文，并在国际会议上发表了论文，获得了学术界的认可和好评。
  • 项目成果：通过这个项目，我学习并掌握了强化学习和计算机自适应测试的相关知识和技术，如强化学习的基本概念，算法，框架，以及题目反应理论，计算机自适应测试的原理，方法，应用等。我也提高了我的算法研发能力，如程序设计，数据处理，代码编写，调试，实验，论文撰写等。该项目提出了一种创新的基于强化学习的题目选择算法，显著改进了计算机自适应测试中的题目选择过程，有效地评估了考生的知识状态，大幅减少了考生需要做的题目数量，达到了项目的预期目标，获得了AIED国际会议的发表。
