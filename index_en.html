Reinforcement Learning Student Ability Testing System

  • Project Introduction: This is a research project on the topic selection algorithm based on reinforcement learning technology, aiming to effectively assess the examinee's knowledge state. The project was carried out to solve two problems existing in the current topic selection algorithm based on information quantity, which are: one is that the information quantity method cannot capture the implicit cognitive information between topics and knowledge points, and the other is that the information quantity method relies on the examinee's estimated knowledge state, which is inaccurate. The project proposes a method of using reinforcement learning technology to automatically learn the topic selection algorithm, which can reasonably capture the cognitive relationship between topics, avoid unnecessary topic tests, and does not rely on the examinee's estimated knowledge state.
  • My Role: In this project, I was the core algorithm developer, responsible for the program design, data processing, code writing and debugging of the entire algorithm part. I worked closely with other team members to complete the project's idea, design, experiment and paper writing.
  • Technology Stack:

  • Main Responsibilities:
    • Responsible for participating in the project's idea discussion and innovation, proposing the idea of using reinforcement learning technology to improve the topic selection process, and designing the algorithm framework and process with team members.
    • Responsible for completing the program design, data processing, code writing and debugging of the entire algorithm part, using the TensorFlow framework to implement the topic selection algorithm based on reinforcement learning, and comparing and analyzing it with the existing topic selection algorithm based on information quantity.
    • Responsible for conducting a large number of experiments using simulated examinee data and real examinee data, verifying the effectiveness and superiority of the algorithm, and using tools such as Numpy, Pandas, Scikit-learn and Matplotlib for data processing and visualization.
    • Responsible for writing academic papers with team members, and publishing papers at international conferences, receiving recognition and praise from the academic community.
  • Project Outcomes: Through this project, I learned and mastered the relevant knowledge and technology of reinforcement learning and computer adaptive testing, such as the basic concepts, algorithms, frameworks of reinforcement learning, and the principles, methods, applications of item response theory, computer adaptive testing, etc. I also improved my algorithm development ability, such as program design, data processing, code writing, debugging, experimentation, paper writing, etc. The project proposed an innovative topic selection algorithm based on reinforcement learning, which significantly improved the topic selection process in computer adaptive testing, effectively assessed the examinee's knowledge state, greatly reduced the number of topics that the examinee needed to do, achieved the project's expected goals, and was published at the AIED international conference.
